# ZhuangPinghua-Homepage

炫酷的个人主页. 开发中...




## 😆 技术栈

基于 `rollup` + `pug` 快速开发简单的静态页面. 开发调试使用 `browser-sync` 进行热更新.




## 🚀 Quick start

样式使用 `dark-sass` . 使用 `vscode` 插件自动编译生成 `css` 文件. 

1. 安装依赖

   ```bash
   yarn
   ```

2. 运行开发环境

   ```bash
   yarn dev
   ```

3. 打包项目

   ```bash
   yarn build
   ```
